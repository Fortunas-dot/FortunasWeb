# Fortunas

A high-end, modern technology company website built with React, TypeScript, and Tailwind CSS.

## Overview

Fortunas is a next-generation technology company focused on building modern applications, AI-driven products, and intelligent digital platforms. This website showcases our capabilities and vision with a clean, premium design aesthetic.

## Technology Stack

- **React 18** - Modern UI library
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Smooth animations and transitions
- **React Router** - Client-side routing

## Design Philosophy

- Ultra-clean, white-first design
- Minimalist layout with strong typography
- High-end tech aesthetic (Apple / OpenAI / Stripe-level quality)
- Subtle animations and smooth transitions
- Lots of whitespace
- Professional and confident tone

## Pages

- **Home** - Hero section with value proposition and CTAs
- **About** - Company overview, mission, and vision
- **What We Do** - Services and capabilities
- **Technology** - AI and technical approach
- **Contact** - Contact information and company details

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

The optimized production build will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```
